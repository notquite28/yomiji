# Yomichi (読み道)

A WaniKani study app for Android, built with React Native and Expo. Named for 読み (reading) + 道 (path) — the path of reading.

The original Tsurukame iOS app in `tsurukame/` serves as the behavior reference.

See `ROADMAP.md` for the implementation checklist and parity plan.

## Getting Started

```sh
npm install
npm start
npm run android
```

## Commands

```sh
npm run typecheck   # tsc --noEmit
npm test            # jest --runInBand
```
